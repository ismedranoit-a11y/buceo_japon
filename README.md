# Buceo Japón

Este proyecto es una página web estática que recrea el contenido del sitio www.buceojapon.com, un sitio de Google Sites sobre buceo en Japón.

## Descripción

La página incluye información sobre el instructor Israel, puntos de buceo cerca de Tokio, y opciones para bucear juntos en Atami. El sitio está dividido en páginas separadas para mejor navegación.

## Estructura

- `index.html`: Página de inicio con enlaces a las secciones.
- `quien-soy.html`: Información sobre Israel, el instructor.
- `puntos-buceo.html`: Descripción de varios puntos de buceo en Japón.
- `buceemos-juntos.html`: Información sobre bucear con Israel en Atami, incluyendo precios aproximados.
- `styles.css`: Hoja de estilos con colores analizados del sitio original (fondo rosa claro para cabeceras, gris claro para cuerpo, etc.).
- `images/`: Carpeta con las imágenes descargadas del sitio original.

## Uso

Abre `index.html` en un navegador web para comenzar. Navega entre las páginas usando el menú de navegación.

## Tecnologías

- HTML5
- CSS3 (colores basados en el esquema del sitio original: #ffadad para cabeceras, #f9f9f9 para fondo, etc.)

## Notas

Las imágenes se descargaron localmente para evitar dependencias externas. Los precios en "Buceemos juntos" son aproximados basados en información general; contacta al original para detalles exactos.
